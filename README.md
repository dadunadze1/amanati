# Delivery Company Dispatcher

Courier and admin dispatch dashboard for managing parcels on a map.

## Run locally

```bash
npm start
```

The app starts on `http://127.0.0.1:5173` by default.

## Project structure

- `backend/server.mjs` - local Node.js API and static file server
- `frontend/` - browser UI
- `frontend/js/` - modular frontend logic

Runtime data is stored under `backend/data/` and is intentionally ignored by git.
